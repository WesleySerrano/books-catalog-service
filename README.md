# Books Catalog Service

## About the project

### Main Goal
Design and implement a cloud-native book commerce 
platform using Java, Go, Kubernetes, Kafka, and PostgreSQL. 
Develop independently deployable services following 
domain-driven design, event-driven architecture, 
and Saga-based distributed transactions.

### This Repo
Covers the service for retrieving the catalog of 
Books that will feed the frontend application that 
will trigger the services for purchasing and renting

### Authentication
The identity layer follows OpenID Connect and can be replaced by managed providers such as AWS Cognito, Auth0, or Okta
